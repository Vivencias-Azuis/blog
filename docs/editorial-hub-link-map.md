# Mapa Operacional de Hubs e Links Internos

Este documento organiza os hubs editoriais principais do `VA_blog`, seus pilares, satelites e os proximos links internos recomendados. O objetivo e continuar o interlinking de forma sistematica, sem depender de memoria ou inserir links aleatorios.

## Regras de uso

- Priorizar links entre paginas com a mesma intencao de busca ou etapa da jornada.
- Cada satelite deve apontar para o seu pilar principal.
- Cada pilar deve apontar para 3 a 6 satelites mais importantes.
- Quando possivel, criar trilhas de leitura em ordem de decisao: diagnostico -> tratamento -> direitos -> acesso.
- Evitar excesso de links repetidos no mesmo bloco.
