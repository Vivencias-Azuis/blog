# Shadows

## Scale
| Token | Value | Uso |
| --- | --- | --- |
| shadow.sm | 0 1px 2px rgba(11, 35, 66, 0.08) | cards leves |
| shadow.md | 0 6px 16px rgba(11, 35, 66, 0.12) | modais pequenos |
| shadow.lg | 0 12px 28px rgba(11, 35, 66, 0.16) | overlays e destaques |

## Regras de uso
- Sombras discretas para evitar cansaco visual
- Evitar usar sombras em listas longas (prefira bordas)
