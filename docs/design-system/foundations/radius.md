# Radius

## Scale
| Token | Size | Uso |
| --- | --- | --- |
| radius.0 | 0 | elementos planos |
| radius.sm | 6px | inputs e chips |
| radius.md | 10px | cards e blocos |
| radius.lg | 16px | modais e destaques |
| radius.pill | 999px | tags e badges |

## Regras de uso
- Evitar misturar radius.md e radius.lg no mesmo componente
- Cards e blocos informativos sempre com radius.md
