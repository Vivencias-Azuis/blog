# Typography

## Objetivo
Legibilidade em leitura longa, com hierarquia clara e tom acolhedor.

## Font stack
- Display/Heading: 'Manrope', 'Helvetica Neue', Arial, sans-serif
- Body: 'Source Serif 4', Georgia, 'Times New Roman', serif
- Mono: 'JetBrains Mono', SFMono-Regular, Menlo, monospace

## Scale (mobile-first)
| Token | Size | Line-height | Uso |
| --- | --- | --- | --- |
| type.display.lg | 36px | 44px | hero e titulo de pagina |
| type.display.md | 28px | 36px | titulo de artigo |
| type.heading.lg | 22px | 30px | H2 |
| type.heading.md | 18px | 26px | H3 |
| type.heading.sm | 16px | 24px | H4 |
| type.body.lg | 18px | 28px | lead e introducao |
| type.body.md | 16px | 26px | corpo principal |
| type.body.sm | 14px | 22px | metadata e captions |
| type.label.md | 14px | 20px | labels e chips |
| type.label.sm | 12px | 18px | helper text |

## Weights
- regular: 400
- medium: 500
- semibold: 600
- bold: 700

## Regras de uso
- Body sempre em serif para leitura longa
- Headings em sans para contraste e escaneabilidade
- Evitar italico em blocos longos
- Limite de largura: 65-75 caracteres por linha
