# Spacing

## Base unit
- 8px como unidade base

## Scale
| Token | Size | Uso |
| --- | --- | --- |
| space.0 | 0 | reset |
| space.1 | 4px | micro gaps |
| space.2 | 8px | padding compacto |
| space.3 | 12px | gaps pequenos |
| space.4 | 16px | padding padrao |
| space.5 | 20px | gaps medios |
| space.6 | 24px | sections pequenas |
| space.8 | 32px | sections medias |
| space.10 | 40px | sections grandes |
| space.12 | 48px | hero/pagina |
| space.16 | 64px | wrappers grandes |

## Regras de uso
- Usar multiplos de 4px apenas em micro-ajustes
- Manter espacamento consistente entre blocos de texto e CTAs
