# Header

## Overview
Navegacao principal e acesso rapido a categorias, com foco em clareza e leitura mobile.

## Usage
- Usar em todas as paginas
- Priorizar busca e categorias-chave
- Evitar mais de 6 itens visiveis no desktop

## Anatomy
- Logo
- Nav primary
- Search
- CTA secundario (opcional)
- Menu mobile

## States
- Default
- Sticky (scroll)
- Mobile expanded

## Do's
- Manter contraste alto entre fundo e texto
- Priorizar links de categoria mais acessadas

## Don'ts
- Nao usar mais de 2 CTAs no header
- Nao esconder busca em desktop

## Accessibility
- Focus visible em todos os links
- Menu mobile navegavel por teclado
- Labels claros para botao de menu

## Design Tokens
- color.bg.brand
- color.text.onBrand
- space.4
- type.label.md
