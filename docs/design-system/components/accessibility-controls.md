# Accessibility Controls

## Overview
Controles de acessibilidade para ajuste de fonte, contraste e navegacao por teclado.

## Usage
- Disponivel em todas as paginas de leitura longa
- Pode ficar no header ou em um floating button

## Anatomy
- Font size toggle
- Contrast toggle
- Keyboard help (opcional)

## States
- Default
- Active

## Do's
- Indicar estado ativo com texto + icone
- Persistir preferencia do usuario

## Don'ts
- Nao esconder controles em menus profundos
- Nao depender de cor como unico feedback

## Accessibility
- Botao com aria-pressed
- Navegacao completa via teclado

## Design Tokens
- color.text.primary
- color.bg.surface
- space.3
- radius.pill
