# Featured Block

## Overview
Bloco de destaque para artigos recomendados, hubs ou campanhas pontuais.

## Usage
- Home, categorias e paginas institucionais
- Usar para 1-2 itens de maior prioridade

## Anatomy
- Label (opcional)
- Title
- Description
- Visual (opcional)
- Link

## Variants
- featured-article
- featured-hub

## Do's
- Destacar apenas conteudo prioritario
- Manter tom informativo e acolhedor

## Don'ts
- Nao usar para anuncios agressivos
- Nao empilhar muitos blocos seguidos

## Accessibility
- Label com texto real (nao apenas cor)
- Link com texto descritivo

## Design Tokens
- color.bg.brand.soft
- color.border.brand
- space.6
- radius.md
