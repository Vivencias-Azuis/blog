# CTA

## Overview
Chamadas para acao para newsletter, download de recurso ou contato.

## Usage
- Pos-conteudo ou sidebar
- 1 CTA principal por pagina

## Anatomy
- Title
- Description
- Action (button)
- Input (opcional)

## Variants
- primary
- secondary
- minimal

## States
- Default
- Hover
- Focus
- Disabled

## Do's
- Texto curto e orientado a beneficio
- Botao com verbo direto

## Don'ts
- Nao usar mais de 1 CTA primario na pagina
- Nao usar linguagem ambigua em temas de saude

## Accessibility
- Labels claros para inputs
- Estados de erro visiveis

## Design Tokens
- color.bg.brand.soft
- color.text.primary
- space.4
- radius.md
