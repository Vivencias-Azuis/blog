# Article Card

## Overview
Card para listagem de artigos com informacao essencial: titulo, excerpt, tempo de leitura e tags.

## Usage
- Listas no blog, home e hubs
- Variante featured para destaques
- Variante compact para listas densas

## Anatomy
- Image (opcional)
- Title
- Excerpt
- Metadata (tempo, tags)
- Link

## Variants
- default
- featured
- compact

## States
- Default
- Hover
- Focus

## Do's
- Limitar excerpt a 2-3 linhas
- Mostrar tempo de leitura sempre que possivel

## Don'ts
- Nao misturar featured e compact na mesma grade
- Nao usar imagem de baixa qualidade

## Accessibility
- Card inteiro clicavel com foco visivel
- Alt text descritivo na imagem

## Design Tokens
- color.bg.surface
- color.text.primary
- space.4
- radius.md
- shadow.sm
