# Form

## Overview
Inputs e formularios para contato, newsletter e pesquisas.

## Usage
- Formularios curtos (3-6 campos)
- Feedback imediato de erro

## Anatomy
- Label
- Input/textarea/select
- Helper text
- Error message
- Submit

## States
- Default
- Focus
- Error
- Disabled
- Success

## Do's
- Labels sempre visiveis
- Explicar porque o dado e solicitado

## Don'ts
- Nao usar placeholder como label
- Nao ocultar erros

## Accessibility
- Associar label e input
- Mensagens de erro com aria-live

## Design Tokens
- color.border.default
- color.state.error
- space.3
- radius.sm
