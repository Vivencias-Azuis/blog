# Article Page

## Overview
Template de artigo com leitura longa, blocos de resumo rapido e navegacao contextual.

## Usage
- Todas as paginas de post
- Inserir blocos de apoio: resumo rapido, FAQ, leia tambem

## Anatomy
- Title
- Lead
- Metadata (data, tempo, categoria)
- Content (MDX)
- Summary block
- FAQ
- Related content

## States
- Default
- Print (opcional)

## Do's
- Usar largura de coluna confortavel
- Priorizar subtitulos a cada 2-4 paragrafos

## Don'ts
- Nao inserir CTAs no meio do conteudo principal
- Nao usar mais de 2 cores no corpo do texto

## Accessibility
- Hierarquia correta de headings
- Links com texto descritivo
- Tabelas com header e caption

## Design Tokens
- type.body.md
- type.display.md
- space.6
- color.text.primary
