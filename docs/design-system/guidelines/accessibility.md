# Accessibility Guidelines

## Principios
- Contraste AA minimo para texto e controles
- Navegacao completa via teclado
- Leitura clara em mobile

## Checklist
- Focus visible em todos os elementos interativos
- Labels sempre visiveis em formularios
- Links com texto descritivo
- Estados de erro anunciados com aria-live
- Respeitar prefers-reduced-motion

## Conteudo sensivel
- Evitar linguagem determinista sobre diagnostico
- Indicar limites do conteudo e buscar fontes confiaveis

## Metricas
- Lighthouse a11y >= 90
- Zero bloqueios criticos em leitores de tela
