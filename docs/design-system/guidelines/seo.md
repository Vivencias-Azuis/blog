# SEO Guidelines

## Estrutura
- H1 unico por pagina
- H2/H3 organizados por topico
- Meta title com 50-60 caracteres
- Meta description com 120-155 caracteres

## Conteudo
- Introducao clara nas primeiras 120 palavras
- Responder perguntas frequentes com FAQ quando relevante
- Links internos para artigos relacionados

## Tecnico
- Imagens com alt text
- Dados estruturados quando aplicavel
- Evitar carrossel pesado em mobile

## Performance
- LCP < 2.5s
- CLS < 0.1
- Otimizar imagens e fontes
