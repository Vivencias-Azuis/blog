# Content Guidelines

## Tom e voz
- Acolhedor, direto e responsavel
- Linguagem simples sem perder precisao
- Evitar promessas absolutas

## Estrutura de artigo
- Lead com 2-3 frases
- Subtitulos a cada 2-4 paragrafos
- Listas com 3-7 itens
- Callouts para pontos-chave

## Microcopy
- CTAs com beneficio claro
- Formularios com explicacao de uso dos dados

## Exemplos
- Bom: "Entenda sinais comuns e quando buscar apoio profissional"
- Evitar: "Este guia resolve todos os problemas"
