🎯 SISTEMA DE TARGETING - Como Funciona
================================================================================

📊 TARGETING POR CATEGORIA:
┌─────────────────────────────────────────────────────────────┐
│  Post com categoria "educacao"                             │
│  ↓                                                         │
│  Sistema busca banners com targeting.categories:           │
│  ["educacao", "dicas"]                                     │
│  ↓                                                         │
│  Exibe: Curso sobre Autismo, Terapia Ocupacional          │
└─────────────────────────────────────────────────────────────┘

📊 TARGETING POR TAGS:
┌─────────────────────────────────────────────────────────────┐
│  Post com tags: ["terapia", "ocupacional"]                │
│  ↓                                                         │
│  Sistema busca banners com targeting.tags:                 │
│  ["terapia", "ocupacional"]                                │
│  ↓                                                         │
│  Exibe: Terapia Ocupacional Especializada                 │
└─────────────────────────────────────────────────────────────┘

📊 TARGETING POR PÁGINA:
┌─────────────────────────────────────────────────────────────┐
│  Usuário está na página "/blog"                            │
│  ↓                                                         │
│  Sistema busca banners com targeting.pages:                │
│  ["blog", "home"]                                          │
│  ↓                                                         │
│  Exibe: Todos os banners configurados para blog            │
└─────────────────────────────────────────────────────────────┘

📊 TARGETING POR POSIÇÃO:
┌─────────────────────────────────────────────────────────────┐
│  Componente AdPosition com position="sidebar"              │
│  ↓                                                         │
│  Sistema busca banners com targeting.positions:            │
│  ["sidebar", "content-bottom"]                             │
│  ↓                                                         │
│  Exibe: Banners configurados para sidebar                  │
└─────────────────────────────────────────────────────────────┘

================================================================================

- Sidebar visível com anúncios

- Banners completos com imagens
- Texto tamanho normal

- Sidebar oculta
- Layout em grid 2 colunas
- Banners adaptados
- Texto redimensionado

📱 MOBILE (< 768px):
- Layout em coluna única
- Banners simplificados
- Texto pequeno
- Botões maiores para touch
